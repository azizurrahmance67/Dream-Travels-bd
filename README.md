#  Dream Travels - Online Bus Ticket Booking System

**Dream Travels** is a professional web-based bus ticket management application. Developed as part of a Computer Science project, this system provides a seamless experience for passengers to book seats and for administrators to manage travel data.

---

## 🌐 Live Demo
Check out the live application here: [https://dream-travels-bd.onrender.com](https://dream-travels-bd.onrender.com)

---

## 🌟 Key Features

* **Seat Reservation:** Interactive seat selection interface for passengers.
* **Real-time Booking:** Instant booking processing with automatic fare calculation.
* **Digital Dashboard:** A clean, organized admin dashboard to track all successful bookings.
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Payment Gateway Demo:** Integrated simulation for payments via bKash, Nagad, and Cards.

## 🚀 How to Run Locally

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/azizurrahmance67/Dream-Travels-bd.git](https://github.com/azizurrahmance67/Dream-Travels-bd.git)
    cd Dream-Travels-bd
    ```

2.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Initialize Database:**
    ```bash
    python init_db.py
    ```

4.  **Launch Application:**
    ```bash
    python app.py
    ```
    Access the app at: `http://127.0.0.1:5000`

---

## 📂 Project Structure
* `app.py`: Main application logic and routing.
* `init_db.py`: Database schema and initial data setup.
* `templates/`: HTML files for different views (Home, Booking, Dashboard).
* `static/`: CSS and Images for UI styling.
* `database.db`: SQLite database file (generated after init).
